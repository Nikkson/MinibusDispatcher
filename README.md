## Minibus Dispatcher
### A system for dispatching minibuses.  
Features:  
* Minibuses
    * View all minibuses
    * Add minibuses to file
    * Edit minibuses
    * Remove minibuses from file
* Routes
    * View all routes
    * Add routes to file
    * Edit routes
    * Remove routes from file
* Assign routes to minibuses
* Calculate fuel needed for a minibus/route combination
